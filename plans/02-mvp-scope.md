# MVP Scope

## V1 Goal

The first version should prove that ActionOS can support a real team workspace:

- Create a workspace.
- Add members.
- Create boards.
- Add and manage work items.
- Assign ownership.
- Run meetings.
- Convert meeting action items into trackable board items.
- Review personal and team work.

## Must Have

### Workspace Foundation

- Workspace profile.
- Workspace members.
- User roles.
- Invitations.
- Basic teams or member groups.
- Workspace-level activity tracking.

### Boards

- Create, rename, archive boards.
- Board groups or sections.
- Board items.
- Custom columns.
- Table view.
- Kanban view.
- Filtering.
- Sorting.
- Search.

### Field Types

Initial custom field types:

- Text
- Long text
- Status
- Priority
- Person
- Date
- Tags
- Checkbox
- Number
- Link

### Board Items

Each item should support:

- Title
- Description
- Assignees
- Watchers
- Status
- Priority
- Due date
- Tags
- Comments
- Checklist
- Activity history
- Related items

### Meetings

Meeting support should include:

- Meeting title.
- Date and time.
- Attendees.
- Agenda items.
- Notes.
- Decisions.
- Action items.
- Linked board items.
- Previous unresolved actions.
- Follow-up summary.

### Meeting-to-Task Flow

This is a key differentiator.

Users should be able to:

- Write meeting notes.
- Mark lines as notes, decisions, or action items.
- Convert action items into board items.
- Assign owners and due dates.
- Keep a link back to the source meeting.

### Dashboards

V1 dashboards:

- My Work.
- Workspace home.
- Overdue items.
- Blocked items.
- Upcoming meetings.
- Recent activity.
- Tasks by status.

### Collaboration

V1 collaboration:

- Comments.
- Mentions.
- Assignment changes.
- Status changes.
- Activity log.
- Basic notifications.

## Should Have

These are valuable if time allows:

- Board templates.
- Meeting templates.
- Recurring meetings.
- Saved views.
- Basic workload view.
- Basic import/export.
- Simple reminder rules.

## Not V1

These should wait until the foundation is solid:

- Full automation builder.
- Timeline view.
- Gantt view.
- Formula fields.
- Advanced reporting.
- Real-time collaborative editing.
- External calendar sync.
- File storage.
- Advanced guest permissions.
- Billing.
- Marketplace integrations.
- AI summaries.

## V1 Success Criteria

ActionOS v1 is successful if a small team can use it for one full work cycle:

- Plan tasks.
- Run meetings.
- Capture decisions.
- Assign follow-up work.
- Track progress.
- Review overdue and blocked work.
- Understand ownership.
